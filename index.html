Compare Encoders

Libraries and frameworks encode ASCII characters differently. The OWASP Enterprise Security API (ESAPI) is the reference implementation for the most comprehensive and secure output encoding/escaping. Using this encoder comparison table, you will see how ESAPI exceeds other framework encoders and native encoders.

Select Encoders

Choose from this list of all ESAPI encoding methods and other popular framework and native encoders. We've chosen a few by default.

  • ESAPI Encode for HTML
    $ESAPI.encoder(). encodeForHTML
  • ESAPI Encode for HTML Attribute
    $ESAPI.encoder(). encodeForHTMLAttribute
  • ESAPI Encode for CSS
    $ESAPI.encoder(). encodeForCSS
  • ESAPI Encode for JavaScript
    $ESAPI.encoder(). encodeForJavaScript
  • ESAPI Encode for Base64
    $ESAPI.encoder(). encodeForBase64
  • ESAPI Encode for URL
    $ESAPI.encoder(). encodeForURL
  • JavaScript HTML Encoder
    .text(value).html()
  • JavaScript URL Escaped (Deprecated)
    escape()
  • JavaScript URI Encoder
    encodeUri()
  • JavaScript URI Encoder for Component
    encodeUriComponent()
